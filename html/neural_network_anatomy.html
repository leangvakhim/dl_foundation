The Mathematics of Learning

1. Linear Transformation

$$Z = W \cdot X + b$$
  • Z = The linear output
  • W = Weights (Connection strength)
  • X = Inputs (from previous layer)
  • b = Bias (Node threshold)

2. Activation Function

$$A = f(Z)$$
  • A = Final Activation output
  • f() = Non-linear function (e.g., Sigmoid, ReLU)
